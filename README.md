## Getting Started

To get a local copy up and running follow these simple example steps.


### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/llyram/RPS-React-Native.git
   ```
2. cd into the working directory & Install npm packages:
   ```sh
   cd RPS-React-Native 
   npm install
   ```
3. Start the Project:
   ```sh
   npm start
   ```
